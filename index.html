Setup

  1. Edit <Rocket League Install Dir>\TAGame\Config\DefaultStatsAPI.ini.
  2. Set PacketSendRate above 0, for example 30.
  3. Keep the default Port=49123, or launch the bridge with RL_STATS_PORT.
  4. Run npm start, open this page from http://127.0.0.1:5173, then press Connect.
